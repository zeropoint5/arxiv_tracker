    App-Aware Response Synthesis for User Reviews

    发布时间：2020-07-31

    作者：Umar Farooq, A. B. Siddique, Fuad Jamour, Zhijia Zhao, Vagelis Hristidis

    中文摘要：

    为了提高应用评分，及时并满意地回应用户评价至关重要，这对于应用的流行和成功至关重要。由于评价的激增，开发者手动回应几乎变得不可能。为了应对这一挑战，最近的研究表明了自动回应生成的可能性。然而，由于训练的评论-回应对是从许多不同的应用中汇总而来的，这些模型生成特定于应用的回应仍然具有挑战性，而特定于应用的回应通常是可取的，因为应用具有不同的功能和关注点。仅仅为每个应用构建一个模型（即使用单个应用的评论-回应对进行训练）来解决这一挑战可能是不够的，因为单个应用有限的评论-回应对，并且这些对通常缺乏回应新评论所需的相关信息。为了实现特定于应用的回应生成，本研究提出了AARSynth：一个应用感知的回应合成系统。AARSynth背后的关键思想是向seq2seq模型添加特定于给定应用的信息。给定一个新的用户评价，它首先检索与该应用最相关的顶级K个评价和最相关的应用描述片段。然后，检索到的信息和新的用户评价被输入到一个融合的机器学习模型中，该模型集成了seq2seq模型和机器阅读理解模型。后者有助于消化检索到的评价和应用描述。最后，融合模型生成一个针对特定应用的定制回应。我们使用来自Google Play的大量评价和回应语料库评估了AARSynth。结果显示，AARSynth在BLEU-4评分上优于最先进的系统22.2%。此外，我们的用户研究表明，与最先进的系统相比，AARSynth在回应质量上产生了统计学上显著的改进。

    一句话总结：

    AARSynth通过融合特定应用信息与seq2seq模型，实现了针对特定应用的自动回应生成，显著提升了回应质量。

    Receptive-Field Regularized CNNs for Music Classification and Tagging

    发布时间：2020-07-27

    作者：Khaled Koutini, Hamid Eghbal-Zadeh, Verena Haunschmid, Paul Primus, Shreyan Chowdhury, Gerhard Widmer

    中文摘要：

    卷积神经网络（CNNs）在音乐信息检索（MIR）任务中已被成功应用，无论是作为端到端模型还是作为更复杂系统的特征提取器。然而，MIR领域仍然主要由基于VGG的CNN架构变体主导，通常与更复杂的模块如注意力机制相结合，以及/或在大数据集上进行预训练等技术。与其他领域相比，深度模型如ResNet在MIR中很少被使用。其中一个主要原因是，正如我们将要展示的，深度CNN在音乐领域的泛化能力不足。在本文中，我们提出了一种基于精心设计的正则化策略，使深度架构如ResNet在音乐相关任务上具有竞争力的方法。特别是，我们分析了最近引入的感知域正则化和Shake-Shake，并表明它们显著提高了深度CNN在音乐相关任务上的泛化能力，并且所得到的深度CNN可以优于当前更复杂的模型，如结合预训练和注意力机制的CNN。我们在两个不同的MIR任务和相应的数据集上展示了这一点，从而为我们提出的深度正则化CNN提供了这些数据集的新基线，这些基线也可以用作未来更复杂方法中的特征提取模块。

    一句话总结：

    本文提出了一种基于正则化策略的深度CNN模型，显著提高了音乐信息检索任务的泛化能力，并可作为未来复杂方法中的特征提取模块。

    K-Shot Contrastive Learning of Visual Features with Multiple Instance Augmentations

    发布时间：2020-07-27

    作者：Haohang Xu, Hongkai Xiong, Guo-Jun Qi

    中文摘要：

    本文提出了一种基于视觉特征的 $K$-Shot 对比学习（KSCL）方法，通过应用多种增强技术来探究单个实例内部的样本变化。该方法旨在结合实例间区分和实例内变化的优点，通过学习判别性特征来区分不同实例，并通过将查询与实例增强样本的变体进行匹配来处理实例内的变化。特别是，对于每个实例，它构建一个实例子空间来模拟 $K$-shot 增强中显著变化因素如何组合形成增强变体的配置。给定一个查询，通过将查询投影到它们的子空间中，检索与实例最相关的变体以预测正实例类别。这推广了现有的对比学习方法，可以将其视为一种特殊的一次性案例。通过特征值分解来配置实例子空间，并且可以通过可微的子空间配置端到端训练嵌入网络。实验结果表明，所提出的 $K$-shot 对比学习方法在性能上优于最先进的无监督方法。

    一句话总结：

    本文提出的 $K$-Shot 对比学习方法通过构建实例子空间和端到端训练嵌入网络，实现了在视觉特征学习上的优越性能。

    Screen Tracking for Clinical Translation of Live Ultrasound Image Analysis Methods

    发布时间：2020-07-13

    作者：Simona Treivase, Alberto Gomez, Jacqueline Matthew, Emily Skelton, Julia A. Schnabel, Nicolas Toussaint

    中文摘要：

    超声（US）成像是最常用的非侵入性成像技术之一。然而，超声图像的获取需要同时引导探头和解读图像，这是一项极具挑战性的任务，需要多年的训练。尽管近年来在检查过程中超声图像分析方面取得了许多进展，但将这些结果转化为临床环境并不容易。我们提出了一种通用框架，用于提取超声图像并叠加分析任务的结果，无需任何物理连接或对超声系统的修改。该方法通过跟踪超声医生视角固定的摄像头屏幕来捕捉超声图像，并将捕获的图像重新格式化为正确的宽高比，平均耗时87.66 ± 3.73毫秒。假设这将使这些检索到的图像能够输入到图像处理管道中，提取有助于改进检查的信息。这些信息最终可以通过例如增强现实（AR）头盔实时投影回超声医生的视野中。

    一句话总结：

    本研究提出了一种无需物理连接的通用框架，用于提取和叠加超声图像分析结果，以改善超声检查过程。

    Relevance Transformer: Generating Concise Code Snippets with Relevance Feedback

    发布时间：2020-07-06

    作者：Carlos Gemmell, Federico Rossetto, Jeffrey Dalton

    中文摘要：

    本文研究了自动代码生成工具，这些工具能够增强程序员的编程能力。尽管许多集成开发环境（IDE）已经集成了简单的代码检索功能，但显式代码生成是一个新兴的研究领域。目前，代码生成被视为一种机器翻译任务，使用基于循环神经网络（RNN）的编码器-解码器架构，这些架构在代码描述对上进行了训练。在这项工作中，我们介绍了用于此任务的现代Transformer架构，并进一步提出了一种名为相关Transformer的新模型，该模型通过伪相关反馈结合外部知识。相关Transformer在解码过程中倾向于生成与现有检索到的代码相似的结果，同时强制执行多样性。我们在多个标准代码生成基准数据集上进行了实验，包括Django、Hearthstone和CoNaLa。结果表明，基于BLEU评估，相关Transformer模型在性能上优于现有方法。相关Transformer模型展示了基于Transformer架构在代码生成中的潜力，并引入了一种在推理过程中结合伪相关反馈的方法。

    一句话总结：

    本文提出了一种名为相关Transformer的代码生成模型，通过结合外部知识和伪相关反馈，在代码生成任务中实现了性能提升。

    Tree-Augmented Cross-Modal Encoding for Complex-Query Video Retrieval

    发布时间：2020-07-06

    作者：Xun Yang, Jianfeng Dong, Yixin Cao, Xun Wang, Meng Wang, Tat-Seng Chua

    中文摘要：

    随着互联网上用户生成视频的快速增长，对基于文本的视频检索系统的需求日益加剧。传统方法主要倾向于基于概念的模式进行检索，使用简单的查询，这些查询通常对语义复杂的复杂查询效果不佳。最近，基于嵌入的模式作为一种流行的方法出现。它旨在将查询和视频映射到一个共享的嵌入空间，其中语义相似的文本和视频彼此更接近。尽管这种方法简单，但它放弃了利用文本查询的句法结构，因此在建模复杂查询时不是最优的。为了便于使用复杂查询进行视频检索，我们提出了一种树增强跨模态编码方法，通过联合学习查询的语言结构和视频的时间表示。具体来说，给定一个复杂的用户查询，我们首先递归地构建一个潜在语义树来结构性地描述文本查询。然后，我们设计了一种树增强查询编码器来推导结构感知的查询表示和一个时间注意力视频编码器来建模视频的时间特征。最后，查询和视频都被映射到一个联合嵌入空间进行匹配和排序。在这种方法中，我们对复杂查询有了更好的理解和建模，从而实现了更好的视频检索性能。在大型视频检索基准数据集上的大量实验证明了我们方法的有效性。

    一句话总结：

    本文提出了一种基于树增强的跨模态编码方法，通过联合学习查询的语言结构和视频的时间表示，实现了对复杂查询的更好理解和建模，从而提高了视频检索的性能。
